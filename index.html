Free Dale Askey web archive

Description

This is a collection of a couple simple bash scripts to archive all of the sites collected by John Dupuis. The main site script is run on a cron job. The pdf collection script is run when new pdfs are added.

The collection can be found here.

Contributing

I'm not an expert here. If you know of a better way to do it, please contribute! It can even be something as simple as adding sites to the site text file.

  1. Fork the repository
  2. Do something awesome!
  3. Submit a pull request explaining what your contribution does
